#### 简单使用`github`

1. create  a new repository

2. git bash

   ```shell
   cd d:
   cd github
   ```

3. git clone

4. 维护

   ```shell
   git add .                   # 当前路径下所有文件，添加到待上传的文件列表中
   git commit -m "注释"         # 将add的文件commit 到仓库
   git push origin master      # 关联到github的远程仓库
   ```
